Coding Skills

HTML

HTML is the base language of the web. I use this language to create the basic format and text for any website that I create. I also use this language to help populate content for Kenexa lessons. There is an option to bypass this and write plain English in a converter although I find it more stable to write clear and structured HTML instead.

CSS

CSS or Cascading Styling Sheet is the main language to style, colour and outline webpages. There are options to style within HTML although this breaks the best practice rule of separating concerns. By styling just in CSS, any change to the styles are done in this one language. Therefore, I use this as my main and only styling language.

Bootstrap

Bootstrap is a CSS library that holds snippets and shortcuts for styling webpages. It can be very useful to style using this language if your knowledge of CSS it not that great. I have used this library to style webpages however I prefer CSS as it allows me more control over certain styling areas.

SCSS

SCSS is another library language for CSS. This language however is not accepted as a main source file and needs to be converted back into CSS before it can be used within a website. SCSS brings to the table the ability to create variables that can be reused throughout the code. It also has a better structure to help show the parent child relationship between the related elements. This language is very useful however the additional layer of converting can add an unnecessary step and I find this language is only useful while maintaining large CSS files.

JavaScript

JavaScript is the most widely used language for interactions within web development. I use this language for almost all interactions within my websites. I generally prefer to use vanilla JavaScript as it allows me greater control of interactions. Just like CSS there is a method to create JavaScript interaction within HTML however separating concerns helps with organisation and editing.

CreateJS

CreateJS is the main language for Adobe Animate. It is a JavaScript library that is built to help with interaction within that software. I use this language to create all interaction and some design aspects within the Adobe Animate software.

ReactJS

ReactJS is a JavaScript library that was created by Facebook. Its focus is to help create small and easily read files that link together. Its design is to try and fix the fact that JavaScript files can get very large and make it increasingly difficult to edit as the interactions grow. I have used this language to link together some basic interaction however I find it would be more useful when it is used for large projects.

jQuery

jQuery is again another JavaScript library. Its purpose is to help with commonly used interactions so that an extensive knowledge of JavaScript is not needed. This can be very useful to create motion and clicking events. It also has a large documented structure so that most of its capabilities can be found rather easily. I have a knowledge of this library however I do prefer to create the interaction using vanilla JavaScript to allow greater control.

C#

C# is the general language that Unity uses. It is a by-product of C++ but developed in a way for it to be easier to learn. I have a basic knowledge of this area to help with control of interactions within Unity.


Design Skills

Adobe Photoshop

Adobe Photoshop is excellent at editing and amending photos/illustrations. I use Adobe Photoshop to help subtly tweak, correct, modify and light images to bring out the best of the asset. This usually means removing items from the background that may distract from the focal point.

Adobe Illustrator

Adobe Illustrator is fantastic at creating illustrations and designs. I use this software generally to create line diagrams and electric schematics. Unlike Adobe Photoshop this software is not intended to edit photos but focus on the creation of lines and shapes.

Adobe Animate

Adobe Animate is an incredible tool that helps with 2D animations. I mainly use Adobe Animate as a HTML canvas which allows me to develop interactive media. I create the interaction with the use of CreateJS (JavaScript library). Ultimately, I publish the output as a HTML object and use the product within Kenexa as an external object.

Adobe After Effects

Adobe After Effects is an effective tool which is designed around the ability to add effects to videos. It can also be used to edit and publish videos however this is not it’s main purpose. I use Adobe After Effects to add lighting and motion effects to video outputs. It is also a useful tool to add motion text and 2D animations to a video clip.

Adobe Premier

Adobe Premier is the main tool for editing and clipping videos and audio. Although there are similarities to Adobe After Effects the use of Adobe Premier to edit videos and audio is far superior. This is because the timeline in Adobe Premier is a better layout for this task. I use this software to help create a structured video sequence and to amend the sound if necessary.

Adobe InDesign

Adobe InDesign is a useful tool that complements both Adobe Photoshop and Adobe Illustrator. It’s main strength however is based on the multi-page layout that helps organise and display all work within one file. This application is vector based just like Adobe Illustrator, so it is not limited to pixels. It also can import work from both Adobe Photoshop and Adobe Illustrator so that the main work can be done in those applications and the print/layout can be handled in this application.

3D Studio Max

3DS Max is a 3-dimensional modelling tool. Its strengths are based on the ability to create and develop 3-dimensional objects in a virtual space. This is useful as it provides the opportunity to recreate real life objects that can be controlled, moved and destroyed without doing it in real life. I have used this software to create replications of working engines and equipment for training purposes. I have also used its sister program 3DS Interactive which is a virtual reality scene developer to develop a virtual hanger with a solid H135 helicopter.

V-Ray

V-Ray is a software that works along with 3DS Max. Its main purpose is to provide a more realistic lighting system within the software. It does this with advance mathematics and by using V-Ray 3DS Max can render realistic and beautiful lighted scenes.

Unity

Unity is one of the largest gaming engines in the world. This software is designed to take in 3D models and scenes from software such as 3DS Max. It then uses software and scripts to control interactive scenes. This is generally used for games however it can be applied as a simulation application. I have used this software to create an interactive area as well as take advantage of its VR platform.
